# Sistema_Gestion_de_Tickets
Proyecto desarrollado en C++ con el objetivo de practicar Programación Orientada a Objetos mediante la implementación de un sistema básico de gestión de tickets para soporte técnico.

---

## Características

- Registro de tickets
- Asociación de tickets con clientes
- Búsqueda de tickets por ID
- Cambio de estado del ticket
- Mostrar tickets abiertos
- Mostrar tickets cerrados
- Estadísticas generales
- Guardado de tickets en archivos
- Carga de tickets desde archivos
- Validación de IDs duplicados
- Validación de clientes existentes

---

## Funcionalidades

### Registrar Ticket

Permite registrar un nuevo ticket seleccionando un cliente existente.

---

### Buscar Ticket

Permite localizar un ticket mediante su ID.

---

### Cambiar Estado

Actualiza el estado de un ticket entre:

- Abierto
- EnProceso
- Cerrado

---

### Estadísticas

Muestra:

- Total de tickets
- Tickets abiertos
- Tickets en proceso
- Tickets cerrados
- Prioridades
- Porcentajes

---

### Persistencia de datos

El sistema permite:

- Guardar tickets en un archivo de texto.
- Recuperar los tickets almacenados al volver a ejecutar el programa.
